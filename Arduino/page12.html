Aoicrie DFPlayer Mini MP3 Player

MP3 проигрыватель с адаптером карты памяти.

  • Aoicrie DFPlayer Mini MP3 Player
  • Схема подключения
  • DFPlayer Mini - это небольшой и недорогой проигрыватель MP3-модулей с упрощенным выводом напрямую на динамик. Модуль может использоваться как автономный модуль с прикрепленной батареей, динамиком и кнопками или использоваться в сочетании с Arduino UNO или любым другим с возможностями RX / TX. DFPlayer отлично интегрирует модуль жесткого декодирования через простой последовательный порт, который поддерживает распространенные аудиоформаты, вы можете воспроизводить назначенную музыку без каких-либо других утомительных базовых операций. В модуль прекрасно интегрировано аппаратное декодирование MP3, WAV, WMA. SPECIFICATION Supported sampling rates (kHz): 8 / 11.025 / 12/16 / 22.05 / 24/32 / 44.1 / 48 , 24-bit DAC output, support dynamic range: 90dB, SNR support: 85dB. Программное обеспечение поддерживает драйвер TF-карты, максимальная поддержка 32G TF-карты; поддержка файловой системы FAT16, FAT32; поддержка 32 г U диска; 64 Мбайт NORFLASH.

int gassensorSPin = 8;//connect S pin to digital 8 pin 
int gassensorAPin = 0;//connect A pin to analog 0 pin

void setup() 
{                
  pinMode(gassensorSPin,INPUT);//initialize the MQ-2 gas sensor S pin as input
  pinMode(gassensorAPin,INPUT);//initialize the MQ-2 gas sensor A pin as input
  Serial.begin(9600);   // opens serial port, sets data rate to 9600 bps
}
void loop() 
{
  if(digitalRead(gassensorSPin)==HIGH){
    Serial.print("Methane gas is not detected.   Analog data:"); //send data to the serial monitor
  }else{
    Serial.print("Methane gas is detected.       Analog data:");   //send data to the serial monitor
  }
  Serial.println(analogRead(0));         //send analog data(A pin) to the serial monitor
  delay(50);
}
