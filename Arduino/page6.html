KY-003 Arduino Hall Magnetic Sensor

Датчик Холла (датчик магнитного поля).

  • KY-003 Arduino Hall Magnetic Sensor
  • Схема подключения
  • KEYES KY-003 Модуль магнитного датчика Холла Arduino - это переключатель, который будет включаться / выключаться при наличии магнитного поля. Магнитный датчик Холла KY-003 состоит из чувствительного переключателя на эффекте Холла 3144EUA-S для работы при высоких температурах, резистора 680 Ом и светодиода. Совместимо с популярными электронными платформами, такими как Arduino и Raspberry Pi.

int led = 13;//LED pin
int sensor = 3; //sensor pin
int val; //numeric variable

void setup()
{
	pinMode(led, OUTPUT); //set LED pin as output
	pinMode(sensor, INPUT); //set sensor pin as input
}

void loop()
{
	val = digitalRead(sensor); //Read the sensor
	if(val == LOW) //when magnetic field is detected, turn led on
	{
		digitalWrite(Led, HIGH);
	}
	else
	{
		digitalWrite(Led, LOW);
	}
}
