KY-017 Mercury Tilt Switch Module

Ртутный датчик наклона.

  • KY-017 Mercury Tilt Switch Module
  • Схема подключения
  • В модуле ртутного переключателя наклона Arduino KY-017 используется маленький ртутный шарик, замыкающий цепь, когда вы наклоняете модуль. Этот модуль состоит из ртутного переключателя, резистора 680 Ом и светодиода, который загорается при обнаружении наклона. Ртутный шарик размыкает / замыкает цепь при вращении модуля.
    Описание
  • Код 
    
int led_pin = 13; // Define the LED interface
int switch_pin = 3; // Definition of mercury tilt switch sensor interface
int val; // Defines a numeric variable

void setup()
{
	pinMode(led_pin, OUTPUT);
	pinMode(switch_pin, INPUT);
}

void loop()
{
	val = digitalRead(switch_pin); // check mercury switch state
	if(val == HIGH)
	{
		digitalWrite(led_pin, HIGH);
	}
	else
	{
		digitalWrite(led_pin, LOW);
	}
}
