KY-018 Photoresistor Module

Модуль фоторезистора, используемый для измерения интенсивности света и определния наличия или отсутствия света.

  • KY-018 Photoresistor Module
  • Схема подключения
  • Модуль фоторезистора Arduino KY-018, используемый для измерения интенсивности света. Он может определить наличие или отсутствие света. Этот модуль состоит из фоторезистора и линейного резистора 10 кОм. Сопротивление фоторезистора будет уменьшаться при наличии света и увеличиваться при его отсутствии. Выход аналоговый и определяет интенсивность света.

int sensorPin = 2; //define analog pin 2
int value = 0; 

void setup() {
	Serial.begin(9600); 
} 

void loop() {
	value = analogRead(sensorPin); 
	Serial.println(value, DEC); // light intensity
								// high values for bright environment
								// low values for dark environment
	delay(100); 
}
